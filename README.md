# Shadow-The-Hedgehog-CSS

Created by Shadorux.

## Usage

Add the stylesheet to your page:

```html
<link rel="stylesheet" href="./shadow-the-hedgehog.css">
```

Core classes:

- `.ultimate-button` for Chaos Control aura with a clean hover lift.
- `.ultimate-lifeform-text` for text with red outer glow.
- `.chaos-badge` for small crimson status marker for labels and tags.
- `.chaos-input` for black input with red stroke.
- `.chaos-alert` for a red-edged message block for warnings or callouts.
- `.chaos-link` for glowing red underline treatment for links.
- `.ark-divider` for thin red section divide.
- `.shadow-stat`, `.shadow-stat-label`, and `.shadow-stat-value` for stat box with red frame and black and red gradient.
- `.dark-ark-gradient` for black-to-grey background with red scanlines.
